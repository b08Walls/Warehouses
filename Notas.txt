FUNCIONAMIENTO:

	Dentro de la interfaz se pueden llevar acabo las siguientes acciones:

	Tablas:
		-Las tablas se pueden mostrar u ocultar presionando el boton que tiene su titulo
		-Las tablas se pueden editar para crear un nuevo producto o un nuevo almacen solo
		se debe de presionar el boton nuevo.
		-Para que los cambios se vean reflejados en la tabla de salida o en la base de datos
		se debera de presionar el boton SAVE/GUARDAR
		-Si se desea verificar que los datos de la base de datos coinciden con los de la GUI
		presionar: REFRESH
		-El boton HIDE, tambien oculta la tabla.
		-En la barra de menu podemos seleccionar las tablas que deseamos visualizar y de igual
		manera podemos salir de la interfaz
